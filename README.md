# SnAkE

Project handled with 'virtualenv'

Basic steps to install 'virtualenv'

    > pip3 install virtualenv
    > cd SnAkE
    > ~/Library/Python/3.7/bin/virtualenv .
    > virtualenv --python=<path to python3>/python3 .
    > source ./bin/activate
    > pip3 install -r requirements.txt
    > cd src/server
    > python3 main.py
